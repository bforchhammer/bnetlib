Service Locator

The Service Locator is used to create every resource specific instance. If you wish to extend bnetlib, simply overwrite a service entry in the Service Locator or create your own Locator using the ServiceLocatorInterface.

After the instantiation is done, the locator will try to inject the Service Locator itself and a Locale object (if set), if the object is Service Locator or Locale aware.

use bnetlib\Locale\Locale;
use bnetlib\ServiceLocator\ServiceLocator;

$locator = new ServiceLocator();
$locator->set('wow.entity.character', 'You\Namespace\Character');
$locator->fromArray(array(
    'wow.entity.guild'     => 'You\Namespace\Guild',
    'wow.entity.character' => 'You\Namespace\Character',
));

$config = $locator->get('wow.config.character', true);

$locator->setLocale(new Locale('de_DE'));

Every Entity class must implement EntityInterface and every configuration class must implement ConfigurationInterface.
